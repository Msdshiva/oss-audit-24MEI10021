## Project Title
Open Source Software Audit using VLC Media Player

---

## Description
This project contains Linux shell scripts developed as part of the Open Source Software Capstone Project.  
The scripts analyze system information, verify VLC Media Player installation, audit directories, analyze logs, and generate an open-source manifesto.

---

## Software Used
VLC Media Player

---

## Scripts Included

1. script1_vlc_system_identity.sh  
Displays system information and open source insights.

2. script2_vlc_package_inspector.sh  
Checks whether VLC Media Player is installed.

3. script3_vlc_disk_permission_audit.sh  
Audits directory permissions and disk usage.

4. script4_vlc_log_analyzer.sh  
Analyzes system logs for VLC related activity.

5. script5_vlc_manifesto_generator.sh  
Generates an open source manifesto file.

---

## How to Run Scripts

Make scripts executable

chmod +x *.sh

Run scripts

./script1_vlc_system_identity.sh  
./script2_vlc_package_inspector.sh  
./script3_vlc_disk_permission_audit.sh  
./script4_vlc_log_analyzer.sh  
./script5_vlc_manifesto_generator.sh
